# Análisis Estadístico_LRPD-3

## Descripción

Este repositorio contiene el análisis estadístico y la documentación del proyecto de investigación titulado **"Psicología preventiva en el ámbito digital: educación sexual para adolescentes mediante redes sociales"**. El objetivo fue analizar los datos obtenidos mediante una encuesta aplicada a padres de familia, con el fin de identificar relaciones entre las variables del estudio.

## Objetivo

Presentar el proceso de análisis de datos, los resultados obtenidos y el informe final correspondiente al proyecto de investigación.

## Contenido del repositorio

- **Base de Datos_Educación Sexual. Grupo 2.xlsx**
  - Base de datos utilizada para el análisis.
  - Incluye tablas de frecuencia, análisis estadísticos y gráficos elaborados en Microsoft Excel.

- **Informe LRPD 3. Grupo 2. Estadística.docx**
  - Informe del análisis estadístico.
  - Contiene la interpretación de los resultados, el ajuste del modelo final y las conclusiones del estudio.
